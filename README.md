This is a full typescript monorepo.

### Frontend
- <a href="https://react.dev/">React</a>
- It uses <a href="https://chakra-ui.com/">Chakra</a> as a default component library and consistant styling.
- React Icons
- TanstackQuery for api requests
- Tanstack File Routing
- Vite

### Backend
- <a href="http://nestjs.com/" target="blank"><img src="https://nestjs.com/img/logo-small.svg" width="14px" alt="Nest Logo" /> NestJS</a>
- <a href="https://www.prisma.io/"> Prisma </a> for Ts global typing, to keep frontend and backend synced.
- Postgres Database
- Swagger API points.
- Automatic updates to Swagger.
